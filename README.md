## ⇁ Usage
![main screen](https://i.imgur.com/95dvNn3.png)
![man page](https://i.imgur.com/B8MmyMh.png)

```
tab    -- toggle focus between input and list
j/k    -- move up and down list
enter(input) -- search
enter(list)  -- open man page
esc/q  -- close man page
ctrl+c -- quit
```
